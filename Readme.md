
## Installation

```bash
	# Clone the repository
	git clone https://github.com/sorb999/angular2FormUsingJSON.git

	# Enter the newly created directory
	cd angular2FormUsingJSON

	# Download all dependencies
	npm install
```

## Usage

```bash
	# Start the server(angular 4)
	npm start
```
```bash
	# Start the server(angular 4)
	ng server --open
```
